title: Who it's for
description: Who benefits from dzl, and what you need to run it.
---

dzl is for **DayZ mod developers on Windows** — from someone packing their first PBO to
people running several mods and server variants at once.

## You'll feel at home if you

- Iterate on Enforce Script or configs and keep hunting for the right log file.
- Pack mods with Addon Builder and have been bitten by "Build Successful" that wasn't.
- Tune a server's loot economy by hand-editing `types.xml` and friends.
- Want an AI agent to drive your build/test loop through a clean tool interface.

## What you need

- **Windows.**
- **DayZ** and **DayZ Tools**, both installed via Steam.
- The **.NET 8 SDK** to build and run dzl.
- Optional, per feature: **steamcmd** (Workshop downloads), the **GitHub CLI** (`gh`), and a
  **Steam Web API key** (Workshop search).
