# Code of Conduct

## Our pledge

We want DayZ Labs (`dzl`) to be a welcoming, harassment-free project for everyone,
regardless of experience level or background. Maintainers and contributors are expected
to treat each other with respect.

## Our standards

Behaviour that helps the community:

- Being kind and patient with other contributors.
- Giving and gracefully accepting constructive feedback.
- Focusing on what is best for the project and its users.

Behaviour that is not acceptable includes personal attacks, demeaning or discriminatory
comments, and any conduct a reasonable person would find inappropriate in a professional
setting.

## Scope

This Code of Conduct applies in all project spaces — issues, pull requests, and
discussions — and whenever someone is representing the project in public.

## Enforcement

If you experience or witness unacceptable behaviour, report it privately to the
maintainer through GitHub's
[private vulnerability reporting](https://github.com/Borcioo/dayz-labs/security/advisories/new),
or by contacting **[@Borcioo](https://github.com/Borcioo)**. Reports are handled
confidentially. Maintainers may remove comments, commits, issues, and pull requests that
violate this Code of Conduct, and may ban repeat offenders.
